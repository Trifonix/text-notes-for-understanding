=== Функции === @

Процедура - это последовательность действий

def hello(): 			# Декларация процедуры
	print('Hello World!')	# Вывод
hello()				# Вызов процедуры

Функция - это процедура, которая принимает и возвращает значения

def sum(a, b):		# Декларация функции с параметрами
	return a + b	# Возврат значения
sum(2, 4)		# Вызов функции с аргументами

Ещё могут быть локальные функции - они объявляются и вызываются внутри другой функции

Параметры - значения, принимаемые при декларации функции
Аргументы - значения, передаваемые в функцию при вызове



=== Параметры функции === *

Функция может принимать позиционные параметры
def func(name, age):

Можно делать необязательные параметры
def func(name = "Neo", age = 28):
Необязательные параметры должны идти после обязательных

Можно делать именованные параметры и передавать значения по их имени в ином порядке
def func(name, age):
	pass
func(age = 28, name = "Neo)

Можно указывать какие именно параметры будут именованными через символ звёздочки
def func(name, *, age, status):
Именованные параметры будут справа от звёздочки

А ещё можно делать позиционные параметры через символ /
def func(name, age, /, status):
Такими будут все параметры слева от слэша

Иногда неизвестно сколько аргументов будет передаваться в параметры
def sum(*numbers):
	result = 0
	for number in numbers:
		result += number
	return result
print(sum(1,2,3,4,5)) # Вывод: 15

=== Функция как тип, параметр и результат другой функции === @

Функцию можно записывать в переменную и вызывать её из неё

def sum(a, b):
	return a + b

make_sum = sum
print(make_sum(1, 4))	# Вывод: 5

Ещё функцию можно передавать в параметр другой функции

def do_operation(a, b, operation):
	result = f"Получилось: {operation(a, b)}"
	return result

def sum(a, b): return a + b
def mul(a, b): return a * b

print( do_operation(1, 4, sum) )	# Вывод: 5
print( do_operation(2, 5, mul) )	# Вывод: 10

А ещё функция может быть возвращаемым значением из другой функции

def sum(a, b): return a + b
def sub(a, b): return a - b
def mul(a, b): return a * b

def what_operation(choice):
	if 1 == choice: return sum
	elif 2 == choice: return sub
	else: return mul

result = what_operation(1)
print(result(2, 2))						# Вывод: 4

result = what_operation(2)
print(result(20, 6))					# Вывод: 14

result = what_operation(3)
print(result(5, 5))						# Вывод: 25

При попытке сложить целочисленного и вещественного значений int и float дадут float

=== Область видимости переменных === @

Переменные могут быть глобальными и локальными. Глобальная - это когда например переменная объявлена наверху, а через функцию к ней идёт обращение. А локальная - это когда переменная объявлена внутри функции. Если определить локальную переменную с тем же именем, которое уже есть у глобальной, то в рамках этой функции будет использоваться локальная переменная. Чтобы поменять значение глобальной переменной изнутри функции используется ключевое слово `global`

name = "Neo"
def func():
	global name
	name = "Smith"
	print(name)			# Вывод: Smith
func()
print(name)				# Вывод: Smith

Ещё есть ключевое слово `nonlocal`, с которым можно изменить значение в нелокальной переменной выше уровнем.

def outer():
    num = 8
    def inner():
        nonlocal num
        num = 42
        print(num)
    inner()			# Вывод: 42
    print(num)		# Вывод: 42
outer()
